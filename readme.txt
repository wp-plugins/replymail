== Description ==

Enhance the threaded comments system of WordPress 2.7.
When someone reply to your comment, send a email to you.

== Installation ==

1. Upload `replyMail` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure enable 'setting' -> 'Discussion Settings' -> 'Enable threaded (nested) comments'
4. Go to 'Setting' -> 'replyMail Setting', change the options.
